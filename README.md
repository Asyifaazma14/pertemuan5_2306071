## Deskripsi Project
Project ini merupakan aplikasi Flutter sederhana yang menampilkan informasi produk berupa **kemeja**.  
Aplikasi terdiri dari dua halaman yaitu **Halaman Utama (HomePage)** dan **Halaman Kedua (SecondPage)**.

Pada **Halaman Utama**, pengguna dapat melihat:
- Gambar produk
- Nama produk
- Harga produk
- Tombol untuk menuju halaman detail

Pada **Halaman Kedua**, pengguna dapat melihat:
- Gambar produk
- Nama produk
- Harga produk
- Deskripsi produk
- Tombol untuk kembali ke halaman utama

Navigasi antar halaman menggunakan **Navigator.pushNamed** dan **Navigator.pop**.
